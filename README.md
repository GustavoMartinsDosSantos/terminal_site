# Gustavo Santos — Terminal Portfolio

Portfólio pessoal em formato de terminal, construído somente com HTML, CSS e JavaScript.

## Executar localmente

Não há etapa de build ou dependências. Sirva a pasta com qualquer servidor estático:

```bash
python3 -m http.server 4173
```

Depois acesse `http://localhost:4173`.

## Comandos

- `help`, `about`, `experience`, `skills`, `education` e `contact`
- `projects` ou `project <nome>`
- `ls`, `cd`, `pwd` e `cat` para navegar pelo filesystem virtual
- `open github`, `open linkedin`, `open email`, `lang pt`, `lang en`, `theme` e `clear`

Tab completa comandos e caminhos; as setas percorrem o histórico; `Ctrl+L` limpa o terminal.
O idioma é detectado automaticamente pelo navegador e a escolha manual fica salva localmente.
